
# 顧客データ管理アプリ

顧客の個人情報、来店に関する情報、支払い総額情報などを管理するためのWebアプリケーションです。

## 主な機能

1.記事一覧表示機能 2.記事投稿機能 3.記事更新機能 4.記事削除機能 5.ユーザーログイン機能 6.ユーザーログアウト機能 7.ページネーション機能  

## 使用技術

PHP,Laravel,Node.js,Vue.js,HTML,CSS,Bootstrap,SQLite
